## 六、关键发现

### 6.1 锁竞争观察

并发写测试 CPU 使用率仅 56.8%，而单线程写测试 CPU 使用率 99.9%。这说明：

- **写锁竞争明显**: 8 个线程竞争单一写锁
- **这是真实场景**: 真实应用中并发写入确实会有锁竞争
- **MemoryStorage 性能**: 内存存储在无竞争时可达 ~175,000 ops/s

### 6.2 Release vs Debug 性能

| 模式 | 速度 | 说明 |
|------|------|------|
| Debug | ~15,000 ops/s | 慢但可调试 |
| Release | ~175,000 ops/s | 快 10 倍以上 |
