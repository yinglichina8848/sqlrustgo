```bash
# 2. 日志目录

# 3. 配置目录

```

### 9.2 网络限制

```bash
# 仅监听 localhost
sqlrustgo --host 127.0.0.1

# 使用防火墙限制端口
iptables -A INPUT -p tcp --dport 5432 -s 10.0.0.0/8 -j ACCEPT
iptables -A INPUT -p tcp --dport 5432 -j DROP
```

### 9.3 容器安全

