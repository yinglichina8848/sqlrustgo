### 2.5 覆盖率检查 (重点)

#### 2.5.1 正确测试命令

**必须使用以下命令** (非常重要):

```bash
# 清理缓存
rm -rf target/tarpaulin/

# 运行覆盖率测试 (必须包含 --workspace --all-features)
cargo tarpaulin --workspace --all-features
```

#### 2.5.2 常见错误

| 错误命令 | 结果 |
|---------|------|
| `cargo tarpaulin` | 覆盖率 ~10% (错误) |
| `cargo tarpaulin --workspace` | 覆盖率 ~65% (部分准确) |
| `cargo tarpaulin --workspace --all-features` | 覆盖率 ~66% (正确) |

**注意**: tarpaulin 存在并发问题，不带 `-p` 或 `--workspace` 参数时会丢失大部分包的覆盖率数据

#### 2.5.3 通过标准

| 阶段 | 目标覆盖率 |
|------|-----------|
| Alpha | ≥50% |
| Beta | ≥65% |
| RC | ≥80% |
| GA | ≥85% |

## 5. 常见问题

### Q1: 覆盖率测试结果不一致

**A**: 确保使用 `cargo tarpaulin --workspace --all-features`，不要省略参数

### Q2: clippy 错误

**A**: 使用 `cargo clippy --fix --workspace` 自动修复大部分问题

### Q3: 性能测试失败

**A**: 确保测试环境稳定，关闭其他占用资源的程序
