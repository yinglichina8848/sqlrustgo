# v2.8.0 三平台 AI 协同发布计划

> **For Hermes:** Use subagent-driven-development skill to implement this plan task-by-task.

**Goal:** 协调 Z440/Z6G4/Mac Mini 三平台 AI Agent，在 2026-05-07 前完成 v2.8.0 RC → GA 正式发布

**Architecture:** 以 Mac Mini 为中央协调节点（Hermes），Z6G4（OpenCode）和 Z440（Hermes）为分布式执行 worker。所有代码变更通过 Gitea PR 提交，Mac Mini Hermes 负责审核合并。三平台通过 Issue #11（META 协作手册）作为统一入口，GBrain 提供治理规则，LLM-Wiki 提供知识库。

**Tech Stack:**
- 代码托管: Gitea v1.22.1 (192.168.0.252:3000) + GitCode/Gitee/GitHub 镜像
- CI/CD: Nomad v1.9.6 双节点（hp-z6g4 + z440），8 并发
- 治理: GBrain (ai-brain) + Hermes Gate v0.3 + Baseline Verification
- AI Agent: Hermes (Mac Mini + Z440), OpenCode (Z6G4), TRAE (Mac Mini)
- 知识: LLM-Wiki (sqlrustgo.wiki) + GBrain (ai-brain)
