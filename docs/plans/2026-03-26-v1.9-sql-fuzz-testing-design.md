## 3. Components

### 3.1 SQL Generator

- **Random DDL Generator**: Generate CREATE/ALTER/DROP TABLE
- **Random DML Generator**: Generate INSERT/UPDATE/DELETE/SELECT
- **Expression Generator**: Generate WHERE, JOIN, ORDER BY, GROUP BY expressions
- **Type-aware Generation**: Handle INTEGER, TEXT, FLOAT, DATE types

### 3.2 Query Oracle

- **TLP (Thread-Level Parity)**: Execute same query in different threads, compare results
- **Pivoted Query Synthesis**: Generate inverse queries to verify correctness
- **Narrowing Oracle**: Check for unexpected NULL/empty results

### 3.3 Bug Reporter

- Capture crash/panic logs
- Record inconsistent query pairs
- Generate minimal reproduction test cases

## 5. Testing

- Unit tests for each generator
- Integration tests with known edge cases
- Continuous fuzzing in CI (limited iterations)

## 6. Success Criteria

- [ ] Generate valid DDL/DML statements
- [ ] Detect semantic inconsistencies
- [ ] Report reproducible bug cases
- [ ] Run 1000+ iterations without crash
