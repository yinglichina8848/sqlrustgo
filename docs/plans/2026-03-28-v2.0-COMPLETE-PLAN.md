## 🎯 战略调整

### 原计划 vs 新计划

| 功能 | 原计划版本 | 新计划版本 | 理由 |
|------|-----------|-----------|------|
| **触发器** | v3.0 | **v2.0 P0** | MySQL 5.0 核心功能，必须有 |
| **全文索引** | v3.0 | **v2.0 P0** | 搜索场景刚需 |

| **JSON 函数** | v3.0 | **v2.0 P2** | NoSQL 场景 |
| **分区表** | v3.0 | **v2.x 后续** | 需要先跑通 1T 单机 |
| **CTE RECURSIVE** | v3.0 | **v2.x 后续** | 复杂 SQL，可以晚点 |
