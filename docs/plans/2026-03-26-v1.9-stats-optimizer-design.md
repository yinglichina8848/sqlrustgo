## 1. Statistics Collector 实现

### 1.1 架构设计

```
StorageEngine
    |
    +-- analyze_table(table) -> TableStats
            |
            +-- ColumnStats (每列)
                    - distinct_count (NDV)
                    - null_count
                    - min_value
                    - max_value
                    - histogram (可选)
```

### 1.2 组件

| 组件 | 职责 | 状态 |
|------|------|------|
| TableStats | 表级统计信息 | 已存在 |
| ColumnStats | 列级统计信息 | 已存在 |
| StatisticsProvider | 统计信息提供者接口 | 已存在 |
| DefaultStatsCollector | 默认收集器实现 | 已存在 |
| ANALYZE SQL 解析 | 解析 ANALYZE TABLE 命令 | 已存在 |
| StorageEngine::analyze_table | 存储层分析接口 | ⬜ 待实现 |

### 1.3 实现计划

1. 在 StorageEngine trait 添加 `analyze_table` 方法
2. 实现 MemoryStorage 和 FileStorage 的 analyze_table
3. 集成到 SQL 执行引擎
4. 添加 ANALYZE TABLE SQL 命令支持

## 2. Optimizer 完善

```
Query Plan
    |
    +-- LogicalPlan (已有)

            +-- PhysicalPlan (已有)
                    |
                    +-- CostModel (已有)

                            +-- SimpleCostModel (待增强)
```

### 2.2 组件

| 组件 | 职责 | 状态 |

## 3. Deadlock Detection 完善

### 3.1 架构设计

```
TransactionManager
    |
    +-- LockManager
            |
            +-- DeadlockDetector
                    |
                    +-- Waits-For Graph
                            |
                            +-- DFS Cycle Detection
```

### 3.2 组件

| 组件 | 职责 | 状态 |
|------|------|------|
| DeadlockDetector | 死锁检测器 | ✅ 已实现 |
| LockManager | 锁管理器 | ✅ 已实现 |
| MVCC | 多版本并发控制 | ⬜ 待完善 |
| RR Isolation | 可重复读隔离级别 | ⬜ 待实现 |

### 3.3 实现计划

1. 集成 DeadlockDetector 到 LockManager
2. 添加事务超时机制
3. 实现 RR 隔离级别
4. 添加 MVCC 可见性判断
