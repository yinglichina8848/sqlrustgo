# v2.7.0 版本计划

> 版本: `v2.7.0`  
> 分支建议: `develop/v2.7.0`  
> 计划窗口: 2026-05 ~ 2026-08  
> 最后更新: 2026-04-19

---

## 一、版本目标

### 1.1 主目标

1. 达到 MySQL 5.7 生产必要能力（SMB 场景）
2. 提供可审计的混合检索能力（全文 + RAG + 图）
3. 支持 GMP 审核系统集成

### 1.2 成果定义

1. 可部署: 有部署、备份、恢复、回滚文档与脚本
2. 可观测: 性能/安全/稳定性有可复现报告
3. 可审核: 检索结果有证据来源和追溯路径

---

## 二、版本范围（In Scope）

1. 事务/WAL/恢复稳定化
2. FK/约束/索引主路径强化
3. qmd-bridge 接入层
4. 统一检索 API（`lex`/`vec`/`graph`/`hybrid`）
5. GMP Top 10 审核查询模板
6. 72h 长稳压测与 RC/GA 发布门禁

---

## 三、不在范围（Out of Scope）

1. 完整 MySQL 5.7 全特性 1:1 兼容
2. 跨地域分布式多活
3. 云原生多租户平台化（归入 v3.x）

---
