# SQLRustGo v2.7.0 变更日志

**版本**: v2.7.0 (生产就绪版本)
**发布日期**: 2026-04-22

---

## 变更概述

v2.7.0 是生产就绪版本，包含 WAL 恢复机制、外键稳定性、备份恢复、统一搜索 API、混合重排序、GMP Top10、审计证据链、性能回归修复和 72 小时稳定性增强等重大功能改进。

---

## 重大变更

### Breaking Changes

| 变更 | 影响 | 迁移建议 |
|------|------|----------|
| WAL 恢复机制升级 | 需迁移 WAL 格式 | 使用 `sqlrustgo-tools migrate-wal` |
| 审计证据链 | 需要配置审计存储 | 见 UPGRADE_GUIDE.md |

## v2.7.0-ga (2026-04-22)

### 新功能

#### 核心功能 (T-01 ~ T-10)

##### T-01 WAL 恢复机制
- 实现 WAL (Write-Ahead Logging) 完整恢复能力
- 支持崩溃后的数据一致性恢复
- 增强 WAL 日志的完整性和可靠性

##### T-02 外键稳定性
- 修复外键约束在并发场景下的不稳定问题
- 增强外键级联操作的正确性
- 提升外键验证性能

##### T-03 备份与恢复
- 实现完整的物理备份和逻辑备份功能
- 支持 point-in-time 恢复
- 增强备份验证机制

##### T-04 QMD Bridge (Query Metadata Bridge)
- 实现查询元数据桥接功能
- 支持跨数据库查询元数据关联
- 增强查询计划缓存

##### T-05 统一搜索 API
- 实现 SQL + 向量混合搜索的统一 API
- 支持多模态查询处理
- 增强搜索结果相关性

##### T-06 混合重排序 (Hybrid Rerank)
- 实现混合搜索重排序功能
- 支持向量相似度与 BM25 的融合排序
- 增强搜索结果精确度

##### T-07 GMP Top10
- 实现 GMP (Global Most Popular) Top10 功能
- 支持热点数据自动识别
- 增强缓存策略

##### T-08 审计证据链
- 实现完整的审计证据链功能
- 支持操作溯源和合规性审计
- 增强审计日志不可篡改性

##### T-09 性能回归修复
- 修复 v2.6.0 引入的性能回归问题
- 优化查询执行计划
- 提升高并发场景下的稳定性

##### T-10 72 小时稳定性
- 增强长时间运行稳定性
- 修复内存泄漏问题
- 提升事务处理可靠性

## 升级路径

### v2.6.0 → v2.7.0

详细升级步骤请参考 [UPGRADE_GUIDE.md](./UPGRADE_GUIDE.md)

```bash
# 1. 备份数据
sqlrustgo-tools backup --output /backup/v2.6.0

# 2. 升级到 v2.7.0
git checkout v2.7.0
cargo build --release

# 3. 迁移 WAL 格式 (如有必要)
sqlrustgo-tools migrate-wal --database mydb

# 4. 配置审计存储 (新增)
# 编辑配置文件，添加 v2.7.0 审计配置

# 5. 验证
sqlrustgo-tools verify --database mydb
```
