### 2.2 核心模块

| 模块 | 职责 | 关键组件 |
|------|------|----------|
| Parser | SQL 解析 | Tokenizer, AST |
| Planner | 查询规划 | LogicalPlan, PhysicalPlan |
| Optimizer | 优化执行 | CBO, Rules |
| Executor | 执行算子 | Scan, Join, Aggregate |
| Storage | 存储引擎 | BufferPool, BTree, Columnar |
| Transaction | 事务管理 | MVCC, WAL |
| WAL | 崩溃恢复 | Write-Ahead Logging |
| Search | 统一检索 | lex, vec, graph, hybrid |

## 3. v2.7.0 核心改进

### 3.1 WAL 崩溃恢复 (T-01)

v2.7.0 实现完整的 WAL 崩溃恢复机制：

- 事务日志预写
- 检查点机制
- 崩溃后重放恢复
- PITR (时间点恢复) 支持

### 3.2 FK 稳定性增强 (T-02)

- 外键约束完整性保证
- 级联删除/更新稳定性
- 死锁检测与避免

### 3.3 统一检索 API (T-05)

支持多种检索模式的统一入口：

| 检索模式 | 说明 |
|----------|------|
| lex | 全文检索 (关键词/BM25) |
| vec | 向量语义检索 |
| graph | 图关系检索 |
| hybrid | 混合检索 (RRF/Linear/Composite) |

### 3.4 审计证据链 (T-08)

防篡改哈希链实现：

- 操作溯源
- 审计日志不可篡改
- 合规性报告生成
