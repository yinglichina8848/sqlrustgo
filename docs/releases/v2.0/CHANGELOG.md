## 3. 模块架构

### 3.1 新增模块

```
sqlrustgo/
├── audit/                  # 审计日志模块
│   ├── logger.rs          # 审计日志记录
│   ├── reporter.rs         # 审计报告生成
│   └── historian.rs        # 历史数据追溯
├── auth/                   # 权限认证模块
│   ├── user.rs            # 用户管理
│   ├── role.rs            # 角色管理
│   ├── permission.rs       # 权限控制
│   └── approval.rs         # 审批流程
├── network/                 # 网络通信模块 (扩展现有)
│   ├── tcp_server.rs       # TCP 服务器
│   ├── grpc_server.rs      # gRPC 接口
│   └── protocol/           # 协议实现
├── backup/                 # 备份增强模块 (扩展现有 storage)
│   ├── incremental.rs      # 增量备份
│   ├── hot_backup.rs       # 热备份
│   └── scheduler.rs        # 备份调度
└── replication/            # 主从复制模块 (v2.1)
    ├── master.rs           # 主节点
    ├── slave.rs            # 从节点
    └── protocol.rs         # 复制协议
```

### 3.2 依赖关系

```
                    ┌─────────────────┐
                    │   network/      │  ← TCP/gRPC 多客户端访问
                    │   auth/         │  ← 用户认证 + 权限控制
                    │   audit/        │  ← 审计日志 + 追溯
                    └────────┬────────┘
                             │
                    ┌────────▼────────┐
                    │   executor/     │  ← 查询执行 + 事务控制
                    │   planner/      │  ← JOIN 优化 + 窗口函数
                    └────────┬────────┘
                             │
┌─────────────┐     ┌────────▼────────┐     ┌─────────────────┐
│ replication │ ──► │   storage/     │ ←── │   backup/       │
│   (v2.1)    │     │   WAL/         │     │   增量/热备份    │
└─────────────┘     │   ACID         │     └─────────────────┘

```
