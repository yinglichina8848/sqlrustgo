> **版本**: 1.4.0
> **发布日期**: 2026-03-18
> **代号**: CBO & Vectorization Ready
> **前置版本**: v1.3.0 (Enterprise Ready)

---

## 一、版本概述

v1.4.0 是**性能增强版本**，专注于实现基于代价的优化器(CBO)和向量化执行基础。

### 主要特性

1. **CBO 成本优化器**
   - 代价模型框架
   - 统计信息集成
   - Join 顺序优化
   - 索引选择优化

2. **Join 算子增强**
   - SortMergeJoin (排序合并连接)
   - NestedLoopJoin (嵌套循环连接)

3. **可观测性增强**
   - Prometheus 指标格式
   - /metrics HTTP 端点
   - Grafana Dashboard 模板
