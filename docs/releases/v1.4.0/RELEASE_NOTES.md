# SQLRustGo v1.4.0 Release Notes

> **版本**: v1.4.0
> **发布日期**: 2026-03-18
> **代号**: 查询优化与执行增强
> **当前阶段**: GA (正式发布)

---

## 一、版本概述

### 1.1 发布类型

| 项目 | 值 |
|------|------|
| 版本号 | v1.4.0 |
| 发布类型 | 查询优化与执行增强 |
| 目标分支 | release/v1.4.0 |
| 开发分支 | develop/v1.4.0 |
| 前置版本 | v1.3.0 (GA) |

### 1.2 核心特性

v1.4.0 专注于提升查询性能，通过 CBO 成本优化器和 SortMergeJoin 算子提供更智能的执行计划。

- **CBO 成本优化器**: 基于代价的查询优化器框架
- **SortMergeJoin**: 新的连接算法
- **统计信息集成**: 从 ANALYZE 获取表统计信息
- **可观测性增强**: Prometheus 指标 + /metrics 端点
