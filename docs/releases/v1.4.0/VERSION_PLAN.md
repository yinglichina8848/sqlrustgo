## 一、版本主题

**查询优化与执行增强**

v1.4.0 专注于提升查询性能，通过 CBO 成本优化器和 SortMergeJoin 算子提供更智能的执行计划。

---

## 二、轨道规划

### 2.1 轨道 A: CBO 成本优化器 (P0)

| ID | 功能 | 描述 | 优先级 | 状态 |
|----|------|------|--------|------|
| CBO-01 | CBO 成本模型基础 | 基于代价的查询优化器框架 | P0 | ⏳ |
| CBO-02 | 统计信息集成 | 从 ANALYZE 获取表统计信息 | P0 | ⏳ |
| CBO-03 | Join 顺序优化 | 基于成本的 Join 重排 | P0 | ⏳ |
| CBO-04 | 索引选择优化 | 智能选择索引 vs 全表扫描 | P0 | ⏳ |

| ID | 功能 | 描述 | 优先级 | 状态 |
|----|------|------|--------|------|
| SMJ-01 | SortMergeJoin | 新的连接算法 (替代 HashJoin) | P0 | ⏳ |
| SMJ-02 | SortMergeJoin 测试 | 单元测试和集成测试 | P0 | ⏳ |
