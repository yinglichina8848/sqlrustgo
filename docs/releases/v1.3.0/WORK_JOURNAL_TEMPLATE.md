# 工作日记模板

> **版本**: v1.3.0  
> **用途**: 记录每日开发进展、问题和思考

---

## 模板说明

本模板用于 **v1.3.0 Craft 阶段** 的日常开发记录。

### 使用方式
1. 复制模板到 `docs/releases/v1.3.0/journal/` 目录
2. 按日期命名: `YYYY-MM-DD.md`
3. 每日填写，保持记录

## 使用指南

### 1. 每日记录

```bash
# 创建今日日记
mkdir -p docs/releases/v1.3.0/journal
cp docs/releases/v1.3.0/WORK_JOURNAL_TEMPLATE.md \
   docs/releases/v1.3.0/journal/$(date +%Y-%m-%d).md
```

### 2. 快速搜索

```bash
# 搜索特定关键词
grep -r "关键词" docs/releases/v1.3.0/journal/

# 查看本周所有记录
ls docs/releases/v1.3.0/journal/2026-03-*.md
```

### 3. 统计汇总

```bash
# 统计本周完成的任务数
grep -c "✅" docs/releases/v1.3.0/journal/2026-03-*.md

# 统计代码变更
git diff --stat --since="2026-03-13" --until="2026-03-20"
```
