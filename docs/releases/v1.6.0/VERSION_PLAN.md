# SQLRustGo v1.6.0 版本计划

> **版本**: 1.0
> **日期**: 2026-03-18
> **阶段**: 规划中

---

## 一、版本概述

### 1.1 版本目标

v1.6.0 是**稳定版**，在 v1.5.0 基础上实现 MVCC 骨架和性能优化，为 v2.0 向量化版本奠定基础。

### 1.2 核心特性

| 优先级 | 特性 | 说明 |
|--------|------|------|
| P0 | MVCC 骨架 | 多版本并发控制 |
| P0 | 事务管理器 | 事务生命周期 |
| P1 | Planner 优化 | 物理计划优化 |
| P1 | 基准测试 | 性能测试套件 |
