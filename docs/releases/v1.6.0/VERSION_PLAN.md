# SQLRustGo v1.6.0 版本计划

> **版本**: v1.6.0
> **发布日期**: TBD
> **代号**: 事务隔离 & 性能优化
> **当前阶段**: Alpha
> **前置版本**: v1.5.0 (GA)

---

## 一、版本概述

### 1.1 版本目标

v1.6.0 是 **L3+ 稳定版**，在 v1.5.0 基础上实现：
- MVCC 事务隔离
- WAL 改进
- 索引增强
- 数据类型扩展
- 性能优化

为 v2.0 分布式架构奠定基础。

### 1.2 核心特性

| 优先级 | 特性 | 说明 |
|--------|------|------|
| P0 | MVCC 事务隔离 | 快照隔离、版本链管理 |
| P0 | WAL 改进 | 并发写入、检查点优化 |
| P1 | 索引增强 | 唯一索引、复合索引 |
| P1 | 性能优化 | 查询缓存、基准测试 |
| P2 | 数据类型 | DATE、TIMESTAMP、BLOB |

### 1.3 架构目标

```
v1.5 (L3+ 持久化) → v1.6 (L3+ 事务隔离) → v2.0 (分布式)
```

## 六、v2.0 前置条件

v1.6.0 完成以下功能后，v2.0 可启动分布式架构：

| 功能 | v1.6 状态 | v2.0 依赖 |
|------|------------|-------------|
| MVCC | ✅ | 必须 |
| 事务隔离 | ✅ | 必须 |
| 性能基准 | ✅ | 必须 |
| 数据类型 | ⏸️ | 建议 |

### 延期至 v1.7.0

- SAVEPOINT (T-06)
- SIMD 优化 (P-04)
- BLOB (D-03)
- BOOLEAN (D-04)
- REPL 增强 (R-01~R-04)
- 全文索引优化 (I-06)
