# SQLRustGo v1.6.0 开发计划

> **版本**: v1.6.0
> **制定日期**: 2026-03-18
> **当前阶段**: Craft
> **目标分支**: develop/v1.6.0

---

## 一、开发目标

v1.6.0 专注于**事务隔离 & 性能优化**，在 v1.5.0 基础上实现完整的事务支持和性能提升。

### 1.1 核心任务

| 任务ID | 功能 | 描述 | 优先级 |
|--------|------|------|--------|
| T-01 | MVCC 骨架 | 快照隔离、版本链管理 | P0 |
| T-02 | 事务管理器 | BEGIN/COMMIT/ROLLBACK | P0 |
| T-03 | 隔离级别 | READ COMMITTED | P0 |
| W-01 | WAL 并发 | 多线程 WAL 写入 | P0 |
| I-03 | 唯一索引 | UNIQUE 约束 | P1 |
| P-01 | 查询缓存 | 结果缓存 | P1 |
| P-03 | 基准测试 | TPC-H/TPC-C | P1 |
| D-01 | DATE 类型 | 日期支持 | P1 |
