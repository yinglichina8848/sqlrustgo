## 6. 常见问题

### Q1: TPC-H 数据生成失败

**A**: 检查 rand crate 版本，确保使用 `rand = "0.8"`

### Q2: SQLite 对比结果不一致

**A**: 检查浮点精度，使用 `abs(a - b) < 0.01` 比较

### Q3: 覆盖率测试结果不一致

**A**: 确保使用 `cargo tarpaulin --workspace --all-features`

### Q4: 死锁测试超时

**A**: 死锁检测可能未触发，检查 victim 选择逻辑

### Q5: 连接池超时

**A**: 检查 semaphore 初始化大小，确保与 max_size 一致
