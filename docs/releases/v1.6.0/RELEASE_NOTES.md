## 一、版本概述

### 1.1 发布类型

| 项目 | 值 |

### 1.2 核心特性

v1.6.0 专注于并发事务控制，在 v1.5.0 基础上实现：

- **MVCC**: 多版本并发控制
- **事务隔离**: READ COMMITTED 隔离级别
- **行级锁**: 读写锁机制
- **死锁检测**: Wait-For Graph 检测
- **WAL 改进**: 并发写入优化
- **索引增强**: 唯一/复合/全文索引
- **数据类型**: DATE、TIMESTAMP 支持
- **查询缓存**: LRU + TTL 缓存
