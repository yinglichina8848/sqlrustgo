## 二、异步多线程并发架构

```
Executor (多线程 async)
 │
 ├─ Transaction Task 1 ──→ LockManager.lock().await
 ├─ Transaction Task 2 ──→ LockManager.lock().await
 └─ Transaction Task N ──→ LockManager.lock().await
                              │
                              ↓
                         DeadlockDetector
                         (阻塞时触发 DFS)
                              │
                              ↓
                         Victim 回滚 + 唤醒
```

### 核心设计

| 模块 | 设计 | 优势 |
|------|------|------|
| LockManager | async RwLock | await 不阻塞线程 |
| DeadlockDetector | 阻塞时 DFS | 低开销，即时检测 |
| MVCC | 无锁读 | 读不阻塞写 |
| Waiters | FIFO + async notify | 公平，无 CPU spin |
