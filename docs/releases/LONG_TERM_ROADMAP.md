# SQLRustGo 版本演化计划 v3.0

> **版本**: v3.0
> **更新日期**: 2026-03-13
> **制定依据**: 综合前期讨论及最新工程评审意见
> **核心原则**: 版本语义化、工程现实性、风险可控、可观测性内建

---

## 一、版本号语义与成熟度目标

| 主版本 | 技术范式 | 成熟度等级 | 核心特征 | 对标系统 |
| --- | --------- | ------------------- | -------------------------- | ------------------ |
| 1.x | 单机基础数据库 | L3 (Mini DBMS) | 火山模型、行存、B+Tree、MVCC、SQL-92 | early PostgreSQL |
| 2.x | 单机高性能分析引擎 | L4 (Analytical DB) | 向量化执行、列存、CBO、并行执行 | DuckDB |
| 3.x | 分布式数据库 | L5 (Distributed DB) | 分布式执行、分片、共识、分布式事务 | TiDB / CockroachDB |

**演进路径**: 1.x（可用）→ 2.x（快）→ 3.x（大）

## 二、版本演化总览（2026–2029+）

### 🔹 v1.x 系列：单机基础数据库（2026–2028）

| 版本 | 代号 | 时间 | 核心目标 | 主要功能改进 | 可观测性增强 |
| --------- | -------------- | ------- | ------------------------------- | ----------------------------------------------------------------------------------------- | ----------------------------- |
| **v1.3** | Executor 稳定化 | 2026-03 | 火山模型执行器打牢基础 | Volcano trait、TableScan/Projection/Filter/HashJoin、测试覆盖率（整体≥65%） | 基础日志、Metrics trait、/health 端点 |
| **v1.4** | SQL 完整版 | 2026-03 | 完整 SQL 查询 + CBO 基础 | GROUP BY/聚合/ORDER BY/LIMIT、CBO 代价模型、SortMergeJoin/NestedLoopJoin、统计信息、可观测性 | Prometheus、/metrics、Grafana |
| **v1.5** | 存储引擎 + 表达式优化 | 2026-07 | 持久化存储与表达式优化 | 页式存储、缓冲池、WAL、表堆、基础统计信息（表行数）、B+Tree 索引、IndexScan、常量折叠增强、表达式简化 | 存储指标（页读写、缓冲池命中率） |
| **v1.6** | MVCC + Planner 优化 | 2026-11 | MVCC 骨架 + 事务管理 + Planner 优化 | MVCC 快照隔离、事务管理器、Planner 优化（Join 重排序）、基准测试、性能调优 | 事务指标（活跃事务、提交/中止） |

**说明**: v1.5-v1.6 为短期规划版本，详细开发计划见 [VERSION_ROADMAP.md](./VERSION_ROADMAP.md)。v1.7-v1.10 为远期版本，待 v1.6 完成后根据实际情况调整。

## 三、关键改进点说明（针对评审意见）

| 评审意见 | 采纳调整 |
| -------------------- | -------------------------------------------------------- |
| Planner 重构太晚 | v1.4 提前引入 LogicalPlan，v1.8 再进一步优化物理计划与规则 |
| 统计信息太晚 | v1.5 增加基础统计（表行数），v1.6 完成完整统计信息 |
| 缺少 Expression Engine | v1.4 明确加入表达式框架（BinaryExpr, ColumnRef, Constant 等），作为独立模块 |
| Transaction 时间低估 | v1.6 只做 MVCC 骨架，v1.9 完成完整并发控制，周期适当拉长（4个月/版本） |
| 增加 v1.10 稳定版本 | 新增 v1.10，聚焦稳定性、基准测试、Bug 修复，为 v2.0 铺路 |
| 慢查询日志 | v1.4 的 Query Profiler 中包含慢查询阈值记录 |
| 分布式作为远期探索 | v3.x 标记为长期研究，不强制时间线 |
