## 测试类型

### 1. 单元测试

- **目标 crate**: 所有 crates
- **覆盖率目标**: ≥80%
- **命令**: `cargo test --all-features`

### 2. 集成测试

- **位置**: `tests/`
- **覆盖场景**: 分布式事务、MVCC、故障恢复

### 3. SQL Corpus 测试

- **目标**: 85%+ 兼容性
- **命令**: `cargo test -p sqlrustgo-sql-corpus`

### 4. E2E 测试

- **位置**: `tests/e2e/`
- **框架**: 内置测试框架

### 5. 性能测试

- **工具**: Sysbench
- **目标**: ≥10K QPS
