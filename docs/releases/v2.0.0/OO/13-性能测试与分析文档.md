## 2. 向量化执行性能

### 2.1 向量化 vs Volcano

```rust
#[cfg(test)]
mod vectorized_performance_tests {
    #[test]
    fn test_filter_vectorized_vs_volcano() {
        let data = create_test_batch(1_000_000);
        
        // Volcano model
        let volcano_start = Instant::now();
        let volcano_result = volcano_filter(&data, predicate.clone());
        let volcano_time = volcano_start.elapsed();
        
        // Vectorized model
        let vectorized_start = Instant::now();
        let vectorized_result = vectorized_filter(&data, predicate.clone());
        let vectorized_time = vectorized_start.elapsed();
        
        println!("Volcano: {:?}, Vectorized: {:?}", volcano_time, vectorized_time);
        assert!(vectorized_time < volcano_time);
    }
}
```

| 操作 | 标量 | SIMD | 加速比 |
|------|------|------|--------|
| Filter | 450ms | 120ms | 3.75x |
| Projection | 200ms | 80ms | 2.5x |
| Hash Join | 800ms | 250ms | 3.2x |
| Aggregation | 300ms | 100ms | 3x |

## 3. 列式存储性能

### 3.1 Parquet 扫描性能

```rust
#[cfg(test)]
mod columnar_performance_tests {
    #[test]
    fn test_parquet_scan_vs_row_scan() {
        let row_result = row_storage.scan("orders").unwrap();
        let columnar_result = columnar_storage.scan("orders", &[0, 1, 2]).unwrap();
        
        // Columnar should be faster for analytical queries
        assert!(columnar_result.time < row_result.time);
    }
    
    #[test]
    fn test_projection_pushdown_effectiveness() {
        // Query only 2 of 100 columns
        let full_scan = storage.scan("orders", &(0..100).collect());
        let projected = storage.scan("orders", &[0, 1]); // Only 2 columns
        
        // Projected should be much faster
        assert!(projected.time < full_scan.time / 10);
    }
}
```

#[cfg(test)]
mod parallel_executor_tests {
    #[test]
    fn test_scaling_with_threads() {
        let sizes = [1, 2, 4, 8, 16];
        let mut times = vec![];
        
        for &threads in &sizes {
            let time = benchmark_parallel_scan(threads, 10_000_000);

## 6. 内存管理性能

### 6.1 Arena/Pool 分配器

```rust
#[cfg(test)]
mod memory_performance_tests {
    #[test]
    fn test_arena_vs_system_allocator() {
        let iterations = 100_000;
        
        // System allocator
        let system_time = measure_allocations(iterations, SystemAllocator);
        
        // Arena allocator
        let arena_time = measure_allocations(iterations, ArenaAllocator);
        
        assert!(arena_time < system_time);
    }
}
```
