## 2. 系统架构概述

### 2.1 架构模式

SQLRustGo v2.0.0 采用**分层架构**结合**分布式微服务架构**：

- **分层架构**: Parser → Planner → Optimizer → Executor → Storage
- **分布式架构**: Coordinator + Participant 模式
- **列式存储**: ColumnChunk → ColumnSegment → ColumnarStorage

## 4. 接口设计

### 4.1 核心 Trait 定义

```rust
pub trait StorageEngine: Send + Sync {
    fn scan(&self, table: &TableRef) -> Result<Box<dyn RecordBatchReader>>;
    fn insert(&self, table: &TableRef, batch: RecordBatch) -> Result<usize>;
    fn delete(&self, table: &TableRef, keys: &[Key]) -> Result<usize>;
    fn update(&self, table: &TableRef, keys: &[Key], batch: RecordBatch) -> Result<usize>;
    fn flush(&self) -> Result<()>;
    fn checkpoint(&self) -> Result<()>;
}

pub trait Executor: Send + Sync {
    fn open(&mut self, ctx: &dyn ExecutionContext) -> Result<()>;
    fn next(&mut self, ctx: &dyn ExecutionContext) -> Result<Option<RecordBatch>>;
    fn close(&mut self, ctx: &dyn ExecutionContext) -> Result<()>;
}

pub trait TransactionManager: Send + Sync {
    fn begin(&self, isolation: IsolationLevel) -> Result<TransactionId>;
    fn commit(&self, txid: TransactionId) -> Result<()>;
    fn rollback(&self, txid: TransactionId) -> Result<()>;
}

pub trait Coordinator: Send + Sync {
    fn begin_distributed(&self, isolation: IsolationLevel) -> Result<TransactionId>;
    fn prepare(&self, txid: TransactionId) -> Result<PrepareResult>;
    fn commit(&self, txid: TransactionId) -> Result<CommitResult>;
    fn rollback(&self, txid: TransactionId) -> Result<()>;
}

pub trait RaftNode: Send + Sync {
    fn append_entry(&self, entry: LogEntry) -> Result<bool>;
    fn vote_request(&self, term: u64, candidate_id: NodeId) -> Result<VoteResult>;
    fn tick(&self);
}
```

## 7. 技术选型

### 7.1 核心技术栈

| 组件 | 技术选型 | 说明 |
|------|----------|------|
| 语言 | Rust | 内存安全、高性能 |
| 解析器 | nom/lalrpop | Rust 原生解析器 |
| 执行引擎 | Volcano + Vectorized | 混合执行模型 |
| 列式存储 | Arrow + Parquet | Apache Arrow 生态 |
| 网络通信 | Tokio + gRPC | 异步运行时 |
| 共识协议 | Raft | 分布式共识 |
| 事务协议 | 2PC | 两阶段提交 |

### 7.2 依赖库

| 库 | 版本 | 用途 |
|----|------|------|
| tokio | 1.x | 异步运行时 |
| arrow | 53.x | 列式数据格式 |
| parquet | 53.x | Parquet 文件支持 |
| grpc | 0.3.x | gRPC 通信 |
| serde | 1.x | 序列化 |
| tempdir | 0.3.x | 临时目录 |
