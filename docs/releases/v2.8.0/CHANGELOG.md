# SQLRustGo v2.8.0 变更日志

**版本**: v2.8.0 (生产化+分布式+安全)
**发布日期**: (待定)

---

## 变更概述

v2.8.0 是 SQLRustGo 的生产化+分布式版本，目标是：

1. 达到 MySQL 5.7 功能覆盖率 92%
2. 提供初步分布式能力 (分区表、主从复制、故障转移、负载均衡)
3. 安全性评分达到 92%

---

## 重大变更

### Breaking Changes

| 变更 | 影响 | 迁移建议 |
|------|------|----------|
| 分区表语法变更 | 需要使用新的 PARTITION BY 语法 | 见 MIGRATION_GUIDE.md |
| 主从复制协议 | 需配置 GTID | 见 DEPLOYMENT_GUIDE.md |

## v2.8.0 新功能

### Phase A: 兼容性增强 (T-11, T-12, T-13)

#### T-11 FULL OUTER JOIN
- 实现完整的 FULL OUTER JOIN 支持
- 基于 Hash 的匹配算法
- 3/3 测试通过

#### T-12 TRUNCATE/REPLACE
- 实现 TRUNCATE TABLE 支持
- 实现 REPLACE INTO 语法
- 自增计数器重置

#### T-13 窗口函数完善
- ROW_NUMBER 窗口函数
- RANK 窗口函数
- DENSE_RANK 窗口函数

### Phase B: 初步分布式能力 (T-23~T-27)

#### T-23 分区表
- Range 分区支持
- List 分区支持
- Hash 分区支持
- Key 分区支持
- 分区裁剪优化

#### T-24 主从复制
- GTID 复制协议
- 半同步复制
- 复制延迟监控

#### T-25 故障转移
- 主节点宕机检测 < 5s
- 自动切换 < 30s
- 数据安全保证

#### T-26 负载均衡
- 轮询策略
- 最少连接策略
- 健康检查

#### T-27 读写分离
- SELECT 路由到从节点
- INSERT/UPDATE 路由到主节点
- 事务正确路由

### Phase C: 性能优化 (T-14~T-16)

#### T-14 SIMD 向量化加速
- 向量操作 SIMD 加速
- 目标加速比 ≥ 2x
- CPU 利用率优化

#### T-15 Hash Join 并行化
- 多线程 Hash Join
- 目标加速比 ≥ 1.5x
- 内存使用优化

#### T-16 查询计划器优化
- CBO 优化器启用
- 统计信息收集
- 优化命中率目标 ≥ 80%

### Phase D: 安全加固 (T-17~T-19)

#### T-17 列级权限控制
- SELECT 列权限
- INSERT 列权限
- UPDATE 列权限
- GRANT/REVOKE 语法

#### T-18 审计告警系统
- 完整审计日志
- JSON 格式
- 查询和过滤

#### T-19 数据加密基础
- AES-256 加密
- 密钥管理
- 性能影响 < 10%

### Phase E: 文档与多语言 (T-20~T-22)

#### T-20 英文错误消息
- 英文错误消息
- MySQL 兼容错误码

#### T-21 英文 API 文档
- REST API 英文文档
- 客户端连接指南

#### T-22 安全加固指南
- 生产部署安全配置
- SSL/TLS 配置
