## 3. 覆盖率评估

> 注: 本报告基于测试用例覆盖分析。由于 cargo-tarpaulin 在 macOS ARM64 环境下的兼容性限制，实际行覆盖率(LOC coverage)未能自动采集。以下为基于测试用例结构的覆盖评估。

### 3.1 核心模块覆盖评估

| 模块 | 测试覆盖评估 | 关键覆盖领域 | 覆盖缺口 |
|------|-------------|-------------|---------|
| parser | 高 | SQL DDL/DML/表达式/聚合/窗口函数 | 部分 MySQL 特有语法 |
| planner | 高 | 查询计划生成/Join 计划评估 | 复杂 Join 优化 |
| executor | 中高 | DDL/DML/SIMD/存储过程 | FULL OUTER JOIN 执行路径 |
| storage | 高 | B+Tree/BufferPool/WAL/Page | 列式存储 |
| transaction | 高 | MVCC/WAL 恢复/2PC | 分布式事务协调 |
| security | 高 | 审计/防火墙/加密/告警/TLS | 列级权限实际执行 |
| network | 中 | TCP 基本通信 | MySQL 协议完整覆盖 |
| vector | 中 | SIMD 操作 | 大规模向量索引 |
| distributed | 高 | 分区/复制/故障转移/读写分离 | 跨节点一致性 |
