## 覆盖率指标说明

| 指标 | 含义 | 测量方式 |
|------|------|----------|
| **Region%** | 基本块覆盖（分支/循环/控制流） | llvm-cov |
| **Func%** | 函数调用覆盖比例 | llvm-cov |
| **Line%** | 执行语句占总语句比例 | llvm-cov |

---

## 执行命令

```bash
# 快速检查单个模块覆盖率
cargo llvm-cov -p sqlrustgo-<crate> --all-features 2>/dev/null | grep "^TOTAL"

# 生成 HTML 报告（可视化分析）
cargo llvm-cov -p sqlrustgo-<crate> --all-features --html --open

# 全量覆盖率
cargo llvm-cov --all-features --html --output-dir ./coverage-report
```
