## 四、性能分析

### 4.1 性能特点

1. **词法与解析比例**: 约 1:1 (简单查询 ~1.3µs + ~1.6µs ≈ 3µs)
2. **批量处理效率**: 批量处理相比单独处理有轻微开销优化
3. **边界情况**: 空输入和纯空白有极高的处理速度 (80-120ns)

### 4.2 性能亮点

- **极快边界处理**: 空输入 35ns，空白输入 63ns
- **高吞吐量**: 简单 SELECT 达 757K ops/s
- **批量效率**: 100条批量处理平均每条 1.29µs
