## 版本概述

v1.2.0 的核心目标是性能优化，支持 100万行级数据处理。

### 核心目标

| 目标 | 说明 | 优先级 |
|------|------|--------|
| 向量化执行 | RecordBatch + ColumnarArray | P0 |
| 统计信息 | TableStats + ColumnStats | P0 |
| 简化 CBO | 成本优化器基础版 | P1 |
| 网络层增强 | 异步服务器完善 | P1 |

### 成功标准

| 指标 | v1.1.0 | v1.2.0 目标 |
|------|--------|-------------|
| 数据规模 | 10万行 | 100万行 |
| 查询延迟 | - | <100ms (简单查询) |
| 测试覆盖率 | 94.18% | ≥90% |
| 成熟度等级 | L3 | L4 准备 |
