## 三、核心模块说明

### 3.1 查询服务层 (QueryService)

**位置**: `src/query/mod.rs`

| 组件 | 类型 | 说明 |
|------|------|------|
|__代码0__| trait | 查询服务抽象，解耦 REPL 和执行引擎 |
|__代码0__|结构体| 本地查询服务实现 |

**职责**:
- 接收 SQL 字符串
- 协调各组件完成查询执行
- 返回查询结果

### 3.2 目录服务层 (Catalog)

**位置**: `src/catalog/mod.rs`

| 组件 | 类型 | 说明 |
|------|------|------|
|__代码0__| trait | 元数据管理抽象接口 |
|__代码0__|结构体| 内存目录实现 |
|__代码0__|结构体| 表元数据 |
|__代码0__|结构体| 列元数据 |

**职责**:
- 管理数据库表结构
- 提供表/列元信息查询

### 3.3 优化器层 (Optimizer)

**位置**: `src/optimizer/`

| 组件 | 类型 | 说明 |
|------|------|------|
|__代码0__| trait | 查询优化抽象 |
|__代码0__| trait | 统计信息提供者 |
| `Plan` |结构体| 优化后的执行计划 |
| `Cost` |结构体| 成本估算模型 |
| `Rule` | trait | 优化规则抽象 |

**职责**:
- 逻辑计划 → 物理计划转换
- 基于成本的优化 (CBO)
- 优化规则管理

### 3.4 执行器层 (Executor)

**位置**: `src/executor/mod.rs`

| 组件 | 类型 | 说明 |
|------|------|------|
|__代码0__| trait | 执行器抽象 |
|__代码0__|结构体| 执行引擎 |
|__代码0__|结构体| 执行结果 |

**职责**:
- 执行物理计划
- 返回结果集

### 3.5 存储层 (StorageEngine)

**位置**: `src/storage/`

| 组件 | 类型 | 说明 |
|------|------|------|
|__代码0__| trait | 存储引擎抽象 |
|__代码0__|结构体| 文件存储实现 |
|__代码0__|结构体| 缓冲池 |
|__代码0__|结构体| B+ 树索引 |

**职责**:
- 数据持久化
- 索引管理
- 页面缓存

## 四、数据流图

```
SQL String
    │
    ▼
┌─────────────────────────────────────────────────────────────────────┐
│                         Lexer                                        │
│                   (src/lexer/)                                      │
│  输入: "SELECT * FROM users WHERE id = 1"                         │
│  输出: Token 序列                                                   │
└─────────────────────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────────────────────┐
│                         Parser                                      │
│                   (src/parser/)                                      │
│  输入: Token 序列                                                   │
│  输出: AST (Statement)                                             │
└─────────────────────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────────────────────┐
│                    LogicalPlan                                      │
│                   (src/planner/)                                    │
│  输入: AST                                                          │
│  输出: LogicalPlan (关系代数树)                                     │
└─────────────────────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────────────────────┐
│                    Optimizer                                        │
│                   (src/optimizer/)                                  │
│  输入: LogicalPlan + Statistics                                     │
│  输出: PhysicalPlan (物理执行计划)                                   │
│  优化规则: 谓词下推、投影裁剪、Join 重排序                          │
└─────────────────────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────────────────────┐
│                    Executor                                        │
│                   (src/executor/)                                   │
│  输入: PhysicalPlan                                                │
│  输出: RecordBatch (结果集)                                        │
└─────────────────────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────────────────────┐
│                    StorageEngine                                    │
│                   (src/storage/)                                    │
│  读取数据页、写入 WAL                                               │
└─────────────────────────────────────────────────────────────────────┘
```
