## v2.6.0 (2026-04-17)

### 新功能

#### SQL 合规性
- **Fully Correlated Subqueries**: 支持完全相关子查询
- **Set Operations**: 完善 UNION、INTERSECT、EXCEPT 支持
- **CASE Expressions**: 完整的 CASE 表达式支持
- **Table Expressions**: 完整的表表达式支持

#### 性能优化
- **SIMD 加速**: 向量化执行引擎全面优化
- **向量化聚合**: 支持向量化 COUNT、SUM、AVG、MIN、MAX
- **SIMD 过滤**: 使用 AVX-512 加速 WHERE 子句
- **SIMD JOIN**: 使用 SIMD 加速 HASH JOIN

#### 稳定性增强
- **MVCC SSI**: 实现 Serializable Snapshot Isolation
- **WAL 完整性**: 增强 WAL 可靠性
- **崩溃恢复**: 改进崩溃恢复机制

#### 工具改进
- **交互式 SQL**: 支持交互式 SQL 工具 `sqlrustgo-sql`
- **基准测试**: TPC-H SF0.1 ~ SF100 完整支持
- **集成测试**: MySQL 5.7 兼容模式集成测试

## 升级路径

### v2.5.0 → v2.6.0

详细升级步骤请参考 [UPGRADE_GUIDE.md](./UPGRADE_GUIDE.md)

### v2.1.0 → v2.6.0

```bash
# 1. 备份数据
sqlrustgo-tools backup --output /backup/v2.5.0

# 2. 升级到 v2.5.0 (如尚未升级)
git checkout v2.5.0
cargo build --release

# 3. 升级到 v2.6.0
git checkout v2.6.0
cargo build --release

# 4. 迁移配置
# 编辑配置文件，添加 v2.6.0 新增配置项

# 5. 验证
sqlrustgo-tools verify --database mydb
```
