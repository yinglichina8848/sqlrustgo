# v2.6.0 发布门禁检查清单（对齐综合测试口径）

> 版本: `v2.6.0`  
> 当前阶段: `Alpha`  
> 更新日期: 2026-04-19

---

## 一、门禁分类

1. `A 类`：必须通过，失败则阻塞发布/阶段晋级
2. `B 类`：应通过，失败需风险审批与整改计划

证据要求（统一）：
1. 执行命令
2. 执行日期
3. commit hash
4. 结果摘要
5. 产物路径（`artifacts/` 或报告文档）
