```rust
// Parser 测试示例
#[test]
fn test_select_parsing() {
    let result = parse("SELECT id, name FROM users");
    assert!(result.is_ok());
    
    let stmt = result.unwrap();
    if let Statement::Select(select) = stmt {
        assert_eq!(select.projection.len(), 2);
    }
}
```

### 7.2 测试报告

```yaml
# .github/workflows/test.yml
name: Tests

on: [push, pull_request]

jobs:
  test:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v2
      - name: Run unit tests
        run: cargo test --lib
      - name: Run integration tests
        run: cargo test --test '*_test'
      - name: Run anomaly tests
        run: cargo test --test anomaly
      - name: Generate coverage
        run: cargo tarpaulin --output -p sqlrustgo
```

## 8. 测试维护

### 8.1 测试命名规范

```
模块_测试类型_描述

例如:
- test_parser_select_parsing
```

### 8.2 测试优先级

| 优先级 | 定义 | 执行频率 |
|--------|------|----------|
| P0 | 核心功能/回归 | 每次提交 |
| P1 | 重要功能 | 每日 |
| P2 | 边界场景 | 每周 |
