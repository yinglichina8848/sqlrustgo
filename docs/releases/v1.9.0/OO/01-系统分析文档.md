## 2. 系统概述

### 2.1 系统背景

SQLRustGo 是一个用 Rust 实现的 SQL-92 兼容数据库管理系统，采用模块化分层架构。v1.9.0 版本定位为"单机生产就绪版"，重点提升系统的稳定性、测试覆盖度和可部署能力。

## 5. 领域模型分析

### 5.1 核心领域类

```uml
@startuml
hide circle

class Table {
  +name: String
  +columns: Vec<Column>
  +indexes: Vec<Index>
  +constraints: Vec<Constraint>
  --
  +create()
  +drop()
  +alter()
}

class Column {
  +name: String
  +data_type: DataType
  +nullable: bool
  +default_value: Option<Value>
  +is_unique: bool
  +references: Option<ForeignKey>
}

class Index {
  +name: String
  +columns: Vec<String>
  +index_type: IndexType
  +is_unique: bool
}

class Constraint {
  +name: Option<String>
  +constraint_type: ConstraintType
  +columns: Vec<String>
}

class Transaction {
  +id: TransactionId
  +isolation_level: IsolationLevel
  +state: TransactionState
  +start_time: Timestamp
  --
  +begin()
  +commit()
  +rollback()
}

class Query {
  +sql: String
  +ast: Statement
  +logical_plan: LogicalPlan
  +physical_plan: PhysicalPlan
  +execution_time: Duration
}

class Session {
  +id: SessionId
  +user: String
  +database: String
  +transaction: Option<Transaction>
  --
  +execute()
  +begin_transaction()
  +commit()
}

Table "1" *-- "n" Column
Table "1" *-- "n" Index
Table "1" *-- "n" Constraint
Column --> Constraint
Table --> Transaction
Session "1" *-- "0..1" Transaction
Query --> Session

@enduml
```
