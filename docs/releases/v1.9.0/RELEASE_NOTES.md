# v1.9.0 Release Notes

## 概述
v1.9.0 是单机生产就绪版本，专注于增强查询优化、并发控制和教学场景支持。

## 发布日期
2026-03-26

## 主要功能

### 1. 查询优化
- **ANALYZE SQL**: 收集表统计信息
- **EXPLAIN ANALYZE**: 执行计划分析
- **CboOptimizer**: 基于成本的优化器
- **Statistics Collector**: 统计信息收集器
- **ColumnStats/TableStats**: 列级和表级统计

### 2. 并发控制
- **DeadlockDetector**: 死锁检测器
- **LockManager 集成**: 死锁检测与锁管理器集成

### 3. 执行引擎增强
- **IndexScan 执行**: 索引扫描执行
- **HashJoin 执行**: Hash Join 执行
- **SortMergeJoin**: 排序合并连接

### 4. 约束支持
- **外键约束**: FOREIGN KEY 约束验证 (PR #831)
- **主键约束**: PRIMARY KEY 约束
- **唯一约束**: UNIQUE 约束
- **NOT NULL 约束**: 非空约束

### 5. 教学场景支持
新增 18 个教学场景测试，覆盖：
- DDL/DML 操作
- 查询优化 (ANALYZE, EXPLAIN, CBO)
- 索引结构 (B+Tree)
- 连接操作 (Hash Join)
- 事务管理
- 并发控制 (死锁检测)
- 约束 (FK)
- 查询缓存

### 6. 性能测试
新增 16 个性能测试，覆盖：
- 批量 INSERT 性能
- 查询优化 (谓词下推、投影)
- 连接池 (基本操作、并发压力)
- 向量化 (RecordBatch、批量操作)
- 索引扫描 vs 顺序扫描
- Hash Join 性能
- ORDER BY 性能
- 混合工作负载
- 并发读取
- 查询缓存命中
