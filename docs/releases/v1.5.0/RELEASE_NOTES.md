> **发布日期**: 待定

> **当前阶段**: Craft

## 一、版本概述

### 1.1 发布类型

| 项目 | 值 |
|------|------|
| 版本号 | v1.5.0 |
| 发布类型 | 向量化执行 & CBO 深度优化 |
| 目标分支 | release/v1.5.0 |
| 开发分支 | develop/v1.5.0 |
| 前置版本 | v1.4.0 (GA) |

### 1.2 核心特性

v1.5.0 专注于性能深度优化，通过向量化执行和 CBO 深度集成实现显著性能提升。

- **向量化 Filter**: SIMD 加速过滤操作
- **IndexScan**: 完整索引支持
- **SortMergeJoin**: 优化的 Join 算法
- **向量化聚合**: SIMD 加速聚合运算
