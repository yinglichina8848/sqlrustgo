# SQLRustGo v1.5.0 Release Notes

> **版本**: v1.5.0
> **发布日期**: 待定
> **代号**: 存储引擎 & 表达式优化
> **当前阶段**: Craft

---

## 一、版本概述

### 1.1 发布类型

| 项目 | 值 |
|------|------|
| 版本号 | v1.5.0 |
| 发布类型 | 存储引擎 & 表达式优化 |
| 目标分支 | release/v1.5.0 |
| 开发分支 | develop/v1.5.0 |
| 前置版本 | v1.4.0 (GA) |

### 1.2 核心特性

v1.5.0 专注于持久化存储，在 v1.4.0 基础上实现持久化存储、索引支持和表达式优化。

- **页式存储**: 磁盘页管理
- **缓冲池**: LRU 内存缓存
- **WAL**: 预写日志
- **表堆存储**: 持久化表数据
- **B+Tree 索引**: 索引结构
- **表达式优化**: 常量折叠增强
