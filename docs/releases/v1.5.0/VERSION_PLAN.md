# SQLRustGo v1.5.0 版本计划

> **版本**: 1.0
> **日期**: 2026-03-18
> **阶段**: Craft (设计规划)

---

## 一、版本概述

### 1.1 版本目标

v1.5.0 是**性能优化深入版本**，在 v1.4.0 基础上进一步提升性能。

### 1.2 核心特性

| 优先级 | 特性 | 说明 |
|--------|------|------|
| P0 | 向量化 Filter | SIMD 加速过滤操作 |
| P0 | IndexScan 完整实现 | 与存储层索引集成 |
| P1 | SortMergeJoin 启用 | 自动选择最优 Join 算法 |
| P1 | 向量化聚合 | SIMD 加速聚合运算 |
| P2 | JIT 编译优化 | 运行时编译优化 |
