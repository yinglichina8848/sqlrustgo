# SQLRustGo v1.5.0 版本计划

> **版本**: 1.0
> **日期**: 2026-03-18
> **阶段**: Craft (设计规划)

---

## 一、版本概述

### 1.1 版本目标

v1.5.0 是**存储引擎版本**，在 v1.4.0 基础上实现持久化存储和索引支持。

### 1.2 核心特性

| 优先级 | 特性 | 说明 |
|--------|------|------|
| P0 | 页式存储 | 磁盘页管理 |
| P0 | 缓冲池 | 内存页缓存 |
| P0 | WAL | 预写日志 |
| P0 | 表堆存储 | 持久化表数据 |
| P1 | B+Tree 索引 | 索引结构 |
| P1 | IndexScan | 索引扫描算子 |
| P1 | 表达式优化 | 常量折叠/表达式简化增强 |
| P1 | 基础统计信息 | 表行数统计 |
