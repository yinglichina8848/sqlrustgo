## 三、开发顺序

### Phase 1: 页式存储 (S-01)

1. 页结构定义 (4KB/8KB/16KB)
2. 页读写接口
3. 页校验 (checksum)

### Phase 2: 缓冲池 (S-02)

1. LRU 淘汰算法实现
2. 缓冲池命中统计
3. 预取策略

### Phase 3: WAL (S-03)

1. WAL 日志格式设计
2. 日志写入/刷盘
3. 崩溃恢复机制

### Phase 4: 表堆存储 (S-04)

1. 表堆数据结构
2. 行存储格式
3. 页面映射管理

### Phase 5: B+Tree 索引 (I-01)

1. B+Tree 节点结构
2. 插入/删除/查找
3. 索引维护

### Phase 6: 表达式优化 (E-01, E-02)

1. Float/String 常量折叠
2. 布尔表达式短路求值
3. 冗余表达式消除

### Phase 7: 统计信息 (ST-01)

1. 表行数统计
2. ANALYZE 命令
3. 统计信息存储
