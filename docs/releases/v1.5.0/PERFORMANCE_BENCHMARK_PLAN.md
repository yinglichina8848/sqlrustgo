## 2. 基准测试任务

### PB-01: 缓冲池命中率基准测试

| 项目 | 内容 |
|------|------|
| 任务ID | PB-01 |
| 依赖 | S-02 缓冲池 |
| 目标 | 验证缓冲池命中率 ≥80% |

**测试场景**:
- 场景1: 1000次重复读取同一页面 (预期命中率 99%)
- 场景2: 顺序扫描10000行，重复3次 (预期命中率 >80%)
- 场景3: 随机访问10000行 (预期命中率 约10%)

**验收标准**:
- 重复读取命中率 ≥95%
- 顺序扫描命中率 ≥80%
- 缓冲池指标正确收集

---

### PB-02: 索引性能对比测试

| 项目 | 内容 |
|------|------|
| 任务ID | PB-02 |
| 依赖 | I-01, I-02 |
| 目标 | 验证 IndexScan 性能显著优于 SeqScan |

**测试场景**:
- 场景1: 点查询 (WHERE id = ?) - IndexScan vs SeqScan
- 场景2: 范围查询 (WHERE id > ? AND id < ?) - IndexScan vs SeqScan
- 场景3: 高选择性查询 (返回 <1% 数据)
- 场景4: 低选择性查询 (返回 >50% 数据)

**验收标准**:
| 场景 | SeqScan 基线 | IndexScan 目标 | 提升 |
|------|-------------|-----------------|------|
| 点查询 | 100% | <10% | ≥10x |
| 范围查询 | 100% | <20% | ≥5x |
| 高选择性 | 100% | <15% | ≥6x |
| 低选择性 | 100% | >80% | IndexScan 不适用 |

### PB-03: WAL 写入/恢复性能测试

| 项目 | 内容 |
|------|------|
| 任务ID | PB-03 |

| 目标 | 验证 WAL 写入和恢复性能 |

**测试场景**:
- 场景1: 1000次 INSERT (1KB 数据/次)
- 场景2: 100次 UPDATE (10KB 数据/次)

**验收标准**:
| 场景 | 目标 |
|------|------|
| 1000次 INSERT | <2s |
| 100次 UPDATE | <1s |
| 崩溃恢复 (1GB) | <5s |
