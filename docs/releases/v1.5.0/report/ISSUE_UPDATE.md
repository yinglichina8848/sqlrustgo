## v1.5.0 版本目标

v1.5.0 是**性能优化深入版本**，核心目标是实现：

1. **向量化执行增强**
   - SIMD 指令优化
   - 批量处理优化
   - 列式存储支持

2. **CBO 深度优化**
   - IndexScan 完整实现
   - SortMergeJoin 启用
   - 代价模型集成

3. **查询优化**
   - 向量化 Filter
   - 向量化聚合
   - JIT 编译优化
