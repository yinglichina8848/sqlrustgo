## 四、SQL 注入防护

### 4.1 参数化查询

所有 SQL 操作使用参数化查询：

```rust
// ✅ 使用参数化查询
let query = "SELECT * FROM users WHERE id = ?";
let stmt = conn.prepare(query)?;
let result = stmt.execute([user_id])?;

// ❌ 避免字符串拼接
// let query = format!("SELECT * FROM users WHERE id = {}", user_id);
```
