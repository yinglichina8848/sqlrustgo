## 一、模版使用说明

### 1.1 强制使用声明

```
❌ 不允许: 自由格式的测试计划
✅ 必须: 基于本文档模版创建
```

### 1.2 测试计划文件命名

```
docs/testing/{PHASE}_TEST_PLAN.md
docs/releases/v{VERSION}/{PHASE}_TEST_PLAN.md
```

示例：
- `docs/testing/ALPHA_TEST_PLAN.md`
- `docs/testing/BETA_TEST_PLAN.md`
- `docs/testing/RC_TEST_PLAN.md`
- `docs/releases/v3.1.0/RC_TEST_PLAN.md`

### 1.3 测试计划类型

| 类型 | 阶段 | 触发条件 |
|------|------|----------|
| ALPHA_TEST_PLAN | Alpha | 进入 Alpha 阶段 |
| BETA_TEST_PLAN | Beta | 进入 Beta 阶段 |
| RC_TEST_PLAN | RC | 进入 RC 阶段 |
| GA_TEST_PLAN | GA | 进入 GA 阶段 |

### 创建时必须检查

```
□ 文件路径符合命名规范

```
□ 测试用例覆盖所有 P0 功能
