## 六、常见问题

### Q1: 项目使用什么技术栈？

**A**: Rust Edition 2024，主要依赖：
- Tokio 异步运行时
- thiserror 错误处理
- serde 序列化

详见 [Cargo.toml](../Cargo.toml)

### Q2: 如何运行测试？

**A**: 
```bash
# 运行所有测试
cargo test --all-features

# 运行覆盖率测试
cargo llvm-cov --all-features --all --workspace --html
```

### Q3: 当前版本有什么限制？

**A**: Alpha 版本限制：
- 无 JOIN 支持
- 无子查询支持
- 无事务嵌套
- 无约束支持（外键、唯一键）
- 无索引自动选择

详见 [CHANGELOG.md](../CHANGELOG.md)

### Q4: 如何提交 PR？

**A**: 
1. Fork 项目
2. 创建特性分支（参考 [分支策略](plans/2026-02-16-branch-strategy.md)）
3. 遵循代码规范
4. 添加测试
5. 提交 PR

详见 [CONTRIBUTING.md](../CONTRIBUTING.md)
