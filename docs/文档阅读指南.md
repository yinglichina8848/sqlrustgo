## 七、常见问题

### Q1: 项目使用什么技术栈？

**A**: Rust Edition 2024，主要依赖：
- Tokio 异步运行时
- thiserror 错误处理
- serde 序列化

详见 [Cargo.toml](../Cargo.toml)

### Q2: 如何运行测试？

**A**: 
```bash
# 运行所有测试
cargo test --all-features

# 运行覆盖率测试
cargo llvm-cov --all-features --all --workspace --html
```

### Q3: 当前版本有什么限制？

**A**: Alpha 版本限制：
- 无 JOIN 支持
- 无子查询支持
- 无事务嵌套
- 无约束支持（外键、唯一键）
- 无索引自动选择

详见 [CHANGELOG.md](../CHANGELOG.md)

### Q4: v1.0 和 v2.0 文档有什么区别？

**A**: 
- **v1.0 文档**：当前版本的实现、评估、改进计划
- **v2.0 文档**：下一版本的架构设计、性能优化、重构规划
