## 二、三阶段学习路径

```
🚗 第一阶段：自动档（第1-6周）
   • AI辅助，一键启动
   • 目标：建立信心，激发兴趣

🕹️ 第二阶段：手动档（第7-12周）
   • 手动操控，理解机理
   • 目标：理解原理，掌握核心

🔧 第三阶段：修车技能（第13-18周）
   • 深入底层，调试优化
   • 目标：培养分析、调试与解决复杂问题的能力
```

---

## 三、使用说明

### 实验前准备

1. 提前阅读当周的操作手册
2. 准备好开发环境
3. 了解实验目标

### 实验中操作

1. 按照手册步骤逐一操作
2. 记录遇到的问题
3. 截图保存关键步骤

### 实验后提交

1. 完成实验报告
2. 提交到指定分支
3. 等待助教检查
