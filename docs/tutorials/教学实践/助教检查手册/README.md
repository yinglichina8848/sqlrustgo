## 检查流程

### 1. 接收学生提交

```bash
# 查看学生分支
git fetch origin
git branch -r | grep experiment/week-XX
```

### 2. 检出学生代码

```bash
git checkout experiment/week-XX-学生学号
```

### 3. 执行检查

按照当周检查清单逐项检查。

### 4. 记录结果

使用 [PR证据链模板](./templates/pr-evidence-template.md) 记录检查结果。

### 5. 反馈评分

使用 [评分模板](./templates/score-template.md) 给出评分。

---

## 检查原则

### 公平公正

- 所有学生使用相同标准
- 评分有据可依
- 及时反馈问题

### 建设性反馈

- 指出具体问题
- 提供改进建议
- 鼓励优秀实践

### 及时响应

- 48小时内完成检查
- 及时回复学生问题
