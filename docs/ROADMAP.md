## 建议的工作顺序

```
第一阶段 (1-2周)
├── 完善 WHERE 子句 (BETWEEN, IN, LIKE, IS NULL)
├── REPL 元命令 (.tables, .schema)
└── 聚合函数 (COUNT, SUM, AVG)

第二阶段 (2-3周)
├── JOIN 查询
├── GROUP BY + ORDER BY + LIMIT
└── 更多数据类型

第三阶段 (3-4周)
├── 复合索引
├── 事务隔离级别
└── 性能优化
```
