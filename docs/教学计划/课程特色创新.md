# AI增强的软件工程 - 课程特色创新

## 一、课程特色

### 1. 真实项目驱动教学
本课程以SQLRustGo项目（Rust数据库系统）为主线，贯穿整个学期的教学。学生通过参与一个真实的、正在开发的开源项目，将软件工程理论付诸实践。

### 2. AI增强的软件开发
引入AI辅助编程工具（TRAE IDE），让学生掌握如何利用AI工具提升软件开发效率，理解AI在软件开发中的优势与局限性。

### 3. 全流程工程实践
从项目草稿 → Alpha版本 → Beta版本 → RC版本 → Release，学生完整经历软件开发的全生命周期，培养工程化思维。

### 4. 团队协作开发
模拟真实企业开发环境，学习Git分支策略、PR流程、代码审核等团队协作技能。

---

## 二、教学创新

### 1. 教学模式创新
- **上半学期**：单一AI工具 + UML建模 + 快速原型（降低难度）
- **下半学期**：协同开发 + 工程治理 + 发布流程（提升能力）

### 2. 考核方式创新
- 过程考核 + 综合考核
- 强调动手能力和团队协作
- 期末大作业采用真实项目开发

### 3. 教学内容与时俱进
- 融入AI辅助编程最新技术
- 结合开源项目实战经验
- 强调工程规范和团队协作

| 创新点 | 具体内容 |
