# 课程目标

## 一、知识目标

1. 掌握软件工程的基本概念、原理和主要开发方法
2. 熟悉软件系统分析与设计的基本理论和方法
3. 了解AI辅助编程工具的原理和使用方法
4. 掌握UML建模的基本技术
5. 理解团队协作开发流程和版本控制规范

## 二、能力目标

1. 能够运用软件工程方法进行需求分析和系统设计
2. 能够使用UML工具进行面向对象的建模
3. 能够使用AI辅助编程工具提升开发效率
4. 能够进行规范的代码编写和单元测试
5. 能够使用Git进行版本管理和团队协作
6. 能够使用CI/CD工具实现自动化构建和部署

## 三、素质目标

1. 培养工程化思维和规范化开发习惯
2. 增强团队协作意识和沟通能力
3. 养成持续学习和自我提升的习惯
4. 树立质量意识和安全意识

---

## 核心项目

本课程以**SQLRustGo项目**为核心实践项目，学生将通过实现一个完整的数据库管理系统来贯穿整个学习过程。
