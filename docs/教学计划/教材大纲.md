# AI增强的软件工程 - 教材大纲（修订版）

## 教材信息

| 项目 | 内容 |
|------|------|
| **教材名称** | AI增强的软件工程 - 提示词工程与智能协作开发 |
| **适用对象** | 计算机科学与技术、软件工程专业本科生 |
| **参考学时** | 64学时（32理论 + 32实训） |
| **编写时间** | 2026-02-22 |
| **核心理念** | AI是增强工具，人类掌握核心思维和方法 |
