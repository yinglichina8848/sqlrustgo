## 核心项目：SQLRustGo

SQLRustGo是一个用Rust语言实现的数据库管理系统，作为本课程的核心实践项目，贯穿整个教学过程。

### 项目目标
实现一个支持SQL子集的轻量级关系型数据库

### 技术栈
- Rust语言
- Git版本控制
- GitHub Actions CI/CD
- AI辅助编程工具（TRAE IDE、Claude Code、GitHub Copilot）

### 阶段性成果
- Week 4: UML设计文档（用例图、类图、顺序图、状态图、架构图）
- Week 6: 模块划分与接口设计文档
- Week 8: Alpha版本（测试覆盖率≥70%）
- Week 10: PR流程实践与项目成熟度评估
- Week 12: 性能优化报告
- Week 14: RC1门禁验收
- Week 16: Release版本与项目展示
