## 核心项目：SQLRustGo

SQLRustGo是一个用Rust语言实现的数据库管理系统，作为本课程的核心实践项目，贯穿整个教学过程。

### 项目目标
实现一个支持SQL子集的轻量级关系型数据库

### 技术栈
- Rust语言
- Git版本控制
- GitHub Actions CI/CD
- AI辅助编程工具

### 阶段性成果
- Week 4: 词法分析器
- Week 6: 语法分析器
- Week 8: Alpha版本
- Week 12: Beta版本
- Week 16: Release版本
