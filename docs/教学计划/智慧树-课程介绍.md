# 课程介绍

## AI增强的软件工程

### 课程简介

"AI增强的软件工程"是软件工程专业的专业必修课程，是指导学生按照工程化、标准化和规范化的方法开发软件系统，并融入人工智能辅助技术的一门综合性课程。

本课程以SQLRustGo项目（Rust数据库系统）为主线，贯穿整个学期的教学。通过将软件工程理论与AI辅助编程工具相结合，培养学生掌握现代软件设计与开发的基本原理、方法和技术。

### 课程特色

1. **真实项目驱动**：以SQLRustGo开源项目为教学载体，学生参与真实的数据库系统开发
2. **AI增强开发**：引入TRAE IDE等AI辅助编程工具，提升开发效率
3. **全流程实践**：从需求分析到版本发布，完整经历软件开发生命周期
4. **团队协作**：模拟企业开发环境，学习Git分支策略、PR流程、代码审核
