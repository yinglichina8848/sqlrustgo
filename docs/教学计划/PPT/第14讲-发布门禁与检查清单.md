## 1.2 门禁的类型

| 门禁类型 | 检查内容 | 多AI中的作用 |
|----------|---------|--------------|
| 代码门禁 | 编译、测试、格式 | 确保基础质量 |
| 质量门禁 | 覆盖率、代码质量 | 保证整体水平 |
| 安全门禁 | 漏洞扫描、依赖检查 | 保障系统安全 |
| 功能门禁 | 集成测试、回归测试 | 验证功能完整 |
| 性能门禁 | 性能测试、压力测试 | 保障用户体验 |

# Part 2: 门禁配置

---

## 2.1 代码门禁

```yaml
# 代码门禁检查
- cargo build --release      # 编译
- cargo test --all-features  # 测试
- cargo clippy -- -D warnings  # 代码质量
- cargo fmt --check         # 格式
```

---

## 2.2 质量门禁

```yaml
# 质量门禁
- 测试覆盖率 ≥ 80%
- 无高危CodeQL警告
- 无安全漏洞
```

---

## 2.3 安全门禁

```yaml
# 安全门禁
- cargo audit 无漏洞
- 依赖无已知CVE
- 代码无SQL注入
- 无硬编码密码
```

# Part 3: RC版本验收

---

## 3.1 RC版本定义

### Release Candidate

- 发布候选版本
- 通过所有门禁
- 准备正式发布

### RC版本特征

- 功能完整
- 质量达标
- 文档完善
- 测试充分

# Part 4: 实践练习

---

## 4.1 配置门禁

```bash
# 创建门禁脚本
cat > scripts/run_gates.sh << 'EOF'
#!/bin/bash
set -e

echo "=== Running Gate Checks ==="

echo "[1/6] Building..."
cargo build --release

echo "[2/6] Running tests..."
cargo test --all-features

echo "[3/6] Running Clippy..."
cargo clippy --all-features -- -D warnings

echo "[4/6] Checking format..."
cargo fmt --check --all

echo "[5/6] Checking coverage..."
cargo tarpaulin --out Xml

echo "[6/6] Running security audit..."
cargo audit

echo "=== All Gates Passed ==="
EOF

chmod +x scripts/run_gates.sh
```

# 核心知识点总结

---

## 1. 门禁的意义

- **终点检验**：多AI协作最终质量保障
- **自动化**：CI/CD自动执行
- **强制通过**：任何一项不通过都不能发布

## 2. 门禁类型

- **代码门禁**：编译、测试、格式
- **质量门禁**：覆盖率、代码质量
- **安全门禁**：依赖、代码安全
