# SQLRustGo 文档目录治理策略

> **版本**: 1.0
> **生效日期**: 2026-04-19
> **维护者**: 文档治理工作组

---

## 1. 文档目录结构

```
docs/
├── index/                  # 总索引与导航（README + 导航页）
├── architecture/           # 架构与设计
├── releases/               # 版本发布文档（vX.Y.Z）
├── governance/             # 流程与治理
├── guides/                 # 教程/操作指南
├── plans/                  # 实施计划（有时效）
├── reports/                # 验证/评估报告（可归档）
└── archive/                # 历史冻结资料
```

## 2. 命名规范

### 2.1 版本目录

| 类型 | 格式 | 示例 | 说明 |
|------|------|------|------|
| 发布版本 | `vX.Y.Z` | `v2.6.0` | 正式发布文档 |
| 规划版本 | `vX.Y` | `v2.0` | 规划阶段文档（仅保留索引） |

- 禁止新建 `vX.Y` 格式目录（历史迁移后只保留索引说明）
- 版本目录使用语义化版本号

### 2.2 子目录

- **必须使用小写**：使用小写字母和连字符
- **禁止混用大小写**：`oo` vs `OO` 不允许
- **避免中文目录名**：优先使用英文

### 2.3 文档文件

| 类型 | 后缀 | 说明 |
|------|------|------|
| 计划文档 | `_PLAN.md` | 实施计划 |
| 报告文档 | `_REPORT.md` | 评估报告 |
| 门禁清单 | `_GATE.md` / `_CHECKLIST.md` | 发布门禁 |
| 设计文档 | `_DESIGN.md` / `_ARCHITECTURE.md` | 技术设计 |

## 3. 索引要求

### 3.1 入口索引

每个目录必须包含 `README.md`，作为该目录的索引页。

### 3.2 主索引

- `docs/README.md` 必须包含所有一级目录的入口链接
- 新增文档必须在对应目录的 `README.md` 中登记

### 3.3 链接校验

- 入口文件（`README.md`, `docs/README.md` 等）链接必须有效
- 使用 `scripts/gate/check_docs_links.sh` 校验

## 4. 生命周期管理

### 4.1 文档状态

| 状态 | 标记 | 说明 |
|------|------|------|
| 活跃 | `[active]` | 正在使用和维护 |
| 已归档 | `[archived]` | 历史文档，不再维护 |

### 4.2 归档规则

| 文档类型 | 归档周期 | 归档位置 |
|----------|----------|----------|
| 计划文档 | 6个月无更新 | `docs/archive/` |
| 报告文档 | 12个月无更新 | `docs/archive/` |
| 发布文档 | 永久保留 | `docs/releases/` |

### 4.3 归档操作

```bash
# 归档脚本示例（待实现）
./scripts/archive_docs.sh --plan 6months
./scripts/archive_docs.sh --report 12months
```

## 5. 门禁规则

### 5.1 CI 门禁

PR 必须通过以下检查：

1. **链接校验**：`scripts/gate/check_docs_links.sh` 通过
2. **入口索引更新**：新增文档已在对应 `README.md` 中登记

### 5.2 PR 检查项

在 PR 描述中新增：

```markdown
### 文档变更
- [ ] 新增文档已在对应目录 README.md 中登记
- [ ] 文档链接校验通过
```

## 6. 可观测指标

### 6.1 每周指标

| 指标 | 计算方式 | 目标 |
|------|----------|------|
| 失效链接数 | `check_docs_links.sh --all` | 0 |
| 未索引文档数 | 文档总数 - 索引登记数 | 0 |
| 重复文件数 | MD5 哈希检测 | 0 |

### 6.2 指标输出

```bash
# 每周生成指标报告
./scripts/metrics/docs_metrics.sh
```

## 7. 违规处理

| 违规类型 | 处理方式 |
|----------|----------|
| 失效链接 | PR 阻塞，要求修复 |
| 未索引文档 | PR 阻塞，要求在 README.md 登记 |
| 命名不规范 | PR 审查时要求重命名 |
