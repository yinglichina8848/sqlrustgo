### 3. 性能对比数据

#### SQLRustGo vs SQLite vs PostgreSQL

| 数据库 | Q1 (聚合) | Q6 (条件聚合) | 备注 |
|--------|------------|---------------|------|
| **SQLRustGo (Mock)** | ~0.96µs | ~1.5µs | 内存模拟，无磁盘I/O |
| **PostgreSQL** | ~1-5ms | ~1-5ms | 真实磁盘I/O |
| **SQLite** | ~5-10ms | ~5-10ms | 真实磁盘I/O |

**注意**: SQLRustGo 使用 MockStorage 内存模拟，PostgreSQL 和 SQLite 执行真实磁盘 I/O 操作。
在相同硬件环境下，SQLRustGo 的内存操作比磁盘数据库快 **1000-10000x**。

## 🔧 测试运行方式

### 运行所有测试

```bash
# SQLRustGo + SQLite 测试
cargo test --test tpch_full_test

# 运行基准测试
cargo test --test tpch_benchmark -- --nocapture

# 运行完整 Q1-Q22 测试
cargo test --test tpch_full_test -- --nocapture
```

### PostgreSQL 测试

```bash
# 设置环境变量
export PGHOST=/tmp
export PGUSER=liying
export PGDATABASE=postgres

# 运行 PostgreSQL 测试（包含 ignored 测试）
cargo test --test tpch_full_test -- --ignored --nocapture
```

### MySQL 测试

```bash
# 设置环境变量
export MYSQL_HOST=localhost
export MYSQL_USER=root
export MYSQL_PASSWORD=secret
export MYSQL_DATABASE=tpch

# 运行 MySQL 测试（包含 ignored 测试）
cargo test --test tpch_full_test -- --ignored --nocapture
```
