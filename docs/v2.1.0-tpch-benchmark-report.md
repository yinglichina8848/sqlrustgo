## 🔧 测试运行方式

### 运行所有测试

```bash
# SQLRustGo + SQLite 测试
cargo test --test tpch_full_test

# 运行基准测试
cargo test --test tpch_benchmark -- --nocapture

# 运行完整 Q1-Q22 测试
cargo test --test tpch_full_test -- --nocapture
```

### PostgreSQL 测试

```bash
# 设置环境变量
export PGHOST=/tmp
export PGUSER=liying
export PGDATABASE=postgres

# 运行 PostgreSQL 测试（包含 ignored 测试）
cargo test --test tpch_full_test -- --ignored --nocapture
```

export MYSQL_HOST=localhost
