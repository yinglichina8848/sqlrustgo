# 当前版本状态

beta/v2.6.0

## 阶段信息

- **阶段**: Beta (外测)
- **开始日期**: 2026-04-20
- **开发分支**: develop/v2.6.0
- **Beta分支**: beta/v2.6.0

## 版本概述

v2.6.0 是 SQLRustGo 迈向 **生产就绪** 的关键版本。

**核心目标**: 达到替代 MySQL 用于简单生产环境的能力。

## 当前执行口径（2026-04-17）

- 当前版本统一为 `alpha/v2.6.0`
- 当前主开发分支为 `develop/v2.6.0`
- 目标: 实现 SQL-92 完整支持 + MVCC SSI 隔离级别
