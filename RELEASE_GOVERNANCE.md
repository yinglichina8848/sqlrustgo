## 版本阶段

```
Draft → Alpha → Beta → RC → GA
```

| 阶段 | 门禁要求 |
|------|----------|
| Draft | 编译通过 |
| Alpha | 测试 ≥ 80% |
| Beta | 测试 ≥ 95%、Clippy 零警告 |
| RC | 测试 100% |
| GA | 所有问题关闭 |
